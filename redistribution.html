Allowed mirrors

World Downloader is a mod initially created by nariol and cubic72, and now mantained by pokechu22 (coremod version) and julialy (litemod version).

It is licensed under the MMPLv2, which, amoung other things, disallows redistribution without permission. The following is a list of allowed locations:

  • The official Minecraft Forums post
  • Pokechu22's github releases section
  • julialy's liteloader version on bintray
  • Soccerguy3's onedrive
  • Tech Geek United's CurseForge mirror

If you want to be added to this list, send pokechu22 a message on the Minecraft Forums.
